Consider the code fragment below:

public class animal {
    private int size;

    int getSize() {
        return size;
    }

    void setSize(int size) {
        this.size = size;
    }

    public static void main (String[] args) {
        System.out.print("Size of animal is " + size);
    }
}

Will Java compile this code?

(1) Yes, size will be initalised to null when declared by the class
(2) No, non-static field ‘size’ cannot be referenced from from static context | 2
